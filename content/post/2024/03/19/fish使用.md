## fish安装

可以使用指令安装 fish

```shell
sudo apt install fish
```

## 将fish设置为默认终端

只需要终端的一行指令

```shell
chsh -s $(which fish)
```

## fish 个性化配置

fish的配置基本上是通过一条指令来的

`fish --config` 之后会打开一个网页，然后在其中可以自己选择好看的主题之类的
